# Анализ изображений и распознавание объектов

Данный проект предназначен для анализа изображений и автоматического поиска объектов на картинках. Программа определяет количество объектов, выделяет их границы, определяет примерную форму (круг или прямоугольник), вычисляет площадь и сохраняет результаты обработки.

Проект использует компьютерное зрение на базе OpenCV для анализа изображений.

Что делает программа

Программа автоматически:

* загружает изображения из папки `images`;
* находит объекты на изображении;
* определяет форму объекта:

  * `Circle` — круглый объект;
  * `Rectangle` — прямоугольный или некруглый объект;
* считает количество найденных объектов;
* вычисляет площадь каждого объекта;
* рисует контуры объектов;
* отображает ограничивающий прямоугольник;
* показывает центр объекта;
* подписывает тип объекта;
* сохраняет обработанные изображения;
* создаёт CSV-отчёт с результатами.

Структура проекта

`images/` — папка с изображениями (датасет). Сюда необходимо загружать картинки для обработки.

`results/` — папка с результатами обработки. После запуска программы здесь появятся обработанные изображения.

`report.csv` — файл с подробной статистикой обработки.

`requirements.txt` — файл со всеми необходимыми библиотеками для запуска проекта.

Поддерживаемые форматы изображений

Программа работает со следующими форматами:

* `.jpg`
* `.jpeg`
* `.png`

Как установить проект

1. Скачайте репозиторий.

2. Убедитесь, что у вас установлен Python.

3. Установите необходимые библиотеки:

```bash
pip install -r requirements.txt
```

Как пользоваться

1. Поместите изображения для анализа в папку `images`.

2. Запустите основной Python-файл:

```bash
python main.py
```

3. После запуска программа начнёт обработку всех изображений из папки `images`.

4. Для каждого изображения программа:

* найдёт объекты;
* выделит их контурами;
* подпишет форму;
* покажет площадь;
* подсчитает статистику.

5. Результаты будут сохранены:

В папке `results` — обработанные изображения.

В файле `report.csv` — подробная статистика.

Что отображается на изображении

Во время обработки программа рисует:

* зелёный контур — круглый объект (`Circle`);
* красный контур — прямоугольный или некруглый объект (`Rectangle`);
* голубой круг — минимальная окружность вокруг объекта;
* фиолетовая точка — центр объекта;
* номер объекта;
* площадь объекта (`S=площадь`).

В левом верхнем углу изображения отображается статистика:

* `Objects` — общее количество объектов;
* `Round` — количество круглых объектов;
* `Rectangle` — количество прямоугольных объектов.

Описание файла report.csv

После обработки создаётся файл `report.csv`.

Он содержит:

* название изображения;
* количество найденных объектов;
* количество круглых объектов;
* количество прямоугольных объектов.

Пример структуры:

Image | Objects | Round Objects | Rectangle Objects

image1.jpg | 5 | 2 | 3

image2.png | 7 | 4 | 3

Как работает определение формы

Программа вычисляет коэффициент округлости объекта.

Если значение больше `0.8`, объект считается круглым (`Circle`).

Если меньше `0.8`, объект определяется как прямоугольный (`Rectangle`).

Возможные ошибки

Ошибка: `Изображения не найдены`

Причина:
В папке `images` нет изображений.

Решение:
Добавьте файлы `.jpg`, `.jpeg` или `.png` в папку `images`.

Ошибка: `Ошибка загрузки`

Причина:
Файл повреждён или не поддерживается.

Решение:
Проверьте изображение и его формат.

Особенности работы

* Все изображения автоматически изменяются до размера `800x600`.
* Очень маленькие объекты (площадь меньше 100 пикселей) игнорируются.
* Обработка выполняется для всех изображений в папке `images`.
* Для остановки обработки можно нажать клавишу `ESC`.

Назначение проекта

Проект может использоваться для:

* анализа изображений;
* задач компьютерного зрения;
* обработки датасетов;
* подсчёта объектов на изображении;
