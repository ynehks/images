## Возможности

| Функция | Описание |
|---------|----------|
|  **Загрузка** | Поддержка JPG, PNG, JPEG |
|  **Обработка** | Изменение размера до 800x600 |
|  **Детекция** | Поиск контуров через Canny |
|  **Классификация** | Круги (roundness > 0.8) |
|  **Визуализация** | Цветные контуры и подписи |
|  **Отчёт** | CSV с результатами |

## Алгоритм

1. Загрузка и изменение размера изображения
2. Преобразование в оттенки серого
3. Размытие по Гауссу (шумоподавление)
4. Детекция границ (Canny)
5. Поиск контуров
6. Расчёт "округлости" (4π × площадь / периметр²)
7. Классификация и отрисовка
8. Сохранение результата и запись в CSV
